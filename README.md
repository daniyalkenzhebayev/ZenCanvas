## Project Vision
Most planners are too rigid. ZenCanvas allows users to create a free-form workspace where tasks, sub-tasks, and habits live together on a single visual plane.

- **Framework:** Flutter
- **Architecture:** Clean Architecture with Feature-driven folders.
