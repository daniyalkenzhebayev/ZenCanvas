# Iteror- Habit & Goal Tracker

Приложение для отслеживания привычек и достижения целей с системой мотивации.

## Возможности

- Календарный трекер привычек с цветовой индикацией
- Система целей с дедлайнами и прогресс-баром
- 8 достижений для мотивации
- Ежедневные мотивационные цитаты
- Статистика: недельные и месячные обзоры
- Заметки для каждого проекта
- Мультипроектность с цветовой кодировкой
- Светлая и тёмная тема

## Установка

Требования: Flutter SDK 3.0+
```bash
git clone https://github.com/YOUR_USERNAME/zen_canvas.git
cd zen_canvas
flutter pub get
flutter run
```
